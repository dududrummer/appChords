# Loops de Percussão

Coloque os arquivos de áudio nesta pasta com o padrão de nome:

`{estilo}_{bpm}.mp3`

## Exemplos de nomes válidos:

| Arquivo                 | Estilo    | BPM |
|-------------------------|-----------|-----|
| `samba_70.mp3`          | Samba     | 70  |
| `samba_80.mp3`          | Samba     | 80  |
| `samba_90.mp3`          | Samba     | 90  |
| `samba_100.mp3`         | Samba     | 100 |
| `samba_110.mp3`         | Samba     | 110 |
| `samba_120.mp3`         | Samba     | 120 |
| `jazz_100.mp3`          | Jazz      | 100 |
| `jazz_120.mp3`          | Jazz      | 120 |
| `jazz_140.mp3`          | Jazz      | 140 |
| `jazz_160.mp3`          | Jazz      | 160 |
| `bossanova_90.mp3`      | Bossa     | 90  |
| `bossanova_100.mp3`     | Bossa     | 100 |
| `bossanova_108.mp3`     | Bossa     | 108 |
| `bossanova_120.mp3`     | Bossa     | 120 |

## Como funciona

O app seleciona automaticamente o arquivo com BPM mais próximo
ao andamento escolhido. Se nenhum arquivo for encontrado para
o estilo, o app usa percussão sintetizada como fallback.

## Formatos suportados

MP3, OGG, WAV, FLAC — qualquer formato suportado pelo browser.
Recomendado: **MP3 a 128kbps** para menor tamanho e ampla compatibilidade.
